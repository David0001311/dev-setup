## Quick methode to add a path to source >> .bashrc
```bash
echo "source YOURPATH" >> ~/.bashrc
```
example:
```bash
echo "source ~/Desktop/My_WorkSpacae/Simulator/ros2_avis/install/setup.bash" >> ~/.bashrc 
```
